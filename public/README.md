# Public Assets Directory

This directory contains static assets for the Keyboard Canvas app.

## Image Files

Place your image files here. The app is currently looking for:
- `note-image.png` - The note paper image that appears to the left of the canvas

## Usage

Images placed in this directory can be referenced in your code using the root path `/`, for example:
- `/note-image.png` will reference `public/note-image.png`

## Supported Formats

- PNG
- JPG/JPEG
- GIF
- SVG
- WebP
